activeadmin-sample
==========
管理画面をさくっと作成できるgemのactive_adminを利用したサンプルアプリです。
ついでにcancancanも利用してユーザ毎に権限管理も行ってみました。

## gemの利用テスト
- active_admin
- cancancan
